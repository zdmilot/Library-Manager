Badge Reference

OEM Verified — The library author is a recognized laboratory automation OEM. The blue checkmark confirms the author name is a protected OEM identity. Packages entering the OEM group must be code-signed with a publisher certificate whose holder name matches the OEM keyword.
Community Library — Published by a third-party developer. No OEM verification badge is shown.
System Library — Built-in Hamilton library shipped with VENUS. Read-only, with verified OEM badge.

Certificate Holder Matching Examples

When a package uses a restricted OEM author name (e.g. "Hamilton"), the code-signing certificate holder must encompass the OEM keyword. Matching is case-insensitive with non-alphanumeric characters stripped.
