REST API: POST /api/publishers/generate-keypair
Library Manager for Venus 6 » REST API Reference » Generate Keypair

Endpoint

POST /api/publishers/generate-keypair

Description

Generates a new Ed25519 key pair for package code signing. Equivalent to the CLI command generate-keypair. Creates a private key PEM, a public key PEM, and a publisher certificate JSON file. By default, the new certificate is automatically registered as trusted.

Parameters

Request Body (application/json)

FieldTypeRequiredDefaultDescription
publisherstringRequired—Publisher name (3–29 characters)
organizationstringNo—Organization name (3–29 characters if provided)
outputDirstringNoCurrent working directoryOutput directory for generated key files
forcebooleanNofalseOverwrite existing key files
noTrustbooleanNofalseDo not automatically register the certificate as trusted
authorPasswordstringNo—OEM author password (required for restricted publisher names)

Response

Returns the standard envelope with data containing the keypair generation result:

Example Response

{
  "success": true,
  "data": {
    "publisher": "John Doe",
    "organization": "Acme Labs",
    "keyId": "abc123def456",
    "fingerprint": "SHA256:abcdef1234567890...",
    "privateKeyPath": "C:\\Keys\\JohnDoe.private.pem",
    "publicKeyPath": "C:\\Keys\\JohnDoe.public.pem",
    "certPath": "C:\\Keys\\JohnDoe.cert.json",
    "trusted": true
  }
}

Examples

cURL

# Generate a keypair with auto-trust
curl -X POST http://localhost:5555/api/publishers/generate-keypair \
  -H "Content-Type: application/json" \
  -d '{
    "publisher": "John Doe",
    "organization": "Acme Labs",
    "outputDir": "C:\\Keys"
  }'

# Generate without auto-trust
curl -X POST http://localhost:5555/api/publishers/generate-keypair \
  -H "Content-Type: application/json" \
  -d '{
    "publisher": "John Doe",
    "outputDir": "C:\\Keys",
    "noTrust": true
  }'

PowerShell

# Generate a keypair with auto-trust
$body = @{
    publisher    = "John Doe"
    organization = "Acme Labs"
    outputDir    = "C:\Keys"
} | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/publishers/generate-keypair `
  -Method POST -ContentType "application/json" -Body $body

# Generate without auto-trust
$body = @{
    publisher = "John Doe"
    outputDir = "C:\Keys"
    noTrust   = $true
} | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/publishers/generate-keypair `
  -Method POST -ContentType "application/json" -Body $body

Error Conditions

  • publisher not provided → 400
  • Publisher name shorter than 3 or longer than 29 characters → 400
  • Organization name shorter than 3 or longer than 29 characters → 400
  • Key files already exist without force: true → 400
  • Output directory not writable → 400
  • Restricted publisher name without authorPassword → 400
