REST API: DELETE /api/libraries/:nameOrId
Library Manager for Venus 6 » REST API Reference » Delete Library

Endpoint

DELETE /api/libraries/:nameOrId

Description

Deletes an installed library. Equivalent to the CLI command delete-lib --yes (confirmation is implicit). By default performs a soft delete (marks the library as deleted in the database but retains the record). Use hard: true for permanent removal. Use keepFiles: true to leave disk files in place.

This endpoint is serialized by an internal mutex to prevent concurrent database writes.

Parameters

Path Parameters

ParameterTypeRequiredDescription
nameOrIdstringRequiredLibrary name or internal database _id

Request Body (application/json)

FieldTypeRequiredDefaultDescription
hardbooleanNofalsePermanently remove the database record (vs soft-delete)
keepFilesbooleanNofalseLeave library and demo method files on disk

Response

Returns the standard envelope with data containing a DeleteResult object:

FieldTypeDescription
libraryNamestringName of the deleted library
deleteTypestring"soft" or "hard"
keepFilesbooleanWhether disk files were retained
comDllsstring[]COM DLLs that should be manually unregistered

Example Response

{
  "success": true,
  "data": {
    "libraryName": "MyLibrary",
    "deleteType": "soft",
    "keepFiles": false,
    "comDlls": []
  }
}

Examples

cURL

# Soft delete (default)
curl -X DELETE http://localhost:5555/api/libraries/MyLibrary

# Hard delete (permanent)
curl -X DELETE http://localhost:5555/api/libraries/MyLibrary \
  -H "Content-Type: application/json" \
  -d '{"hard": true}'

# Soft delete but keep files on disk
curl -X DELETE http://localhost:5555/api/libraries/MyLibrary \
  -H "Content-Type: application/json" \
  -d '{"keepFiles": true}'

PowerShell

# Soft delete (default)
Invoke-RestMethod -Uri http://localhost:5555/api/libraries/MyLibrary `
  -Method DELETE

# Hard delete (permanent)
$body = @{ hard = $true } | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/libraries/MyLibrary `
  -Method DELETE -ContentType "application/json" -Body $body

# Soft delete but keep files on disk
$body = @{ keepFiles = $true } | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/libraries/MyLibrary `
  -Method DELETE -ContentType "application/json" -Body $body

Error Conditions

  • Library not found → 404
  • Library already deleted (soft) → 400
