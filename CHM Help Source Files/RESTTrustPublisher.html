REST API: POST /api/publishers/trust
Library Manager for Venus 6 » REST API Reference » Trust Publisher

Endpoint

POST /api/publishers/trust

Description

Trusts or revokes a publisher certificate. Equivalent to the CLI command trust-publisher. When trusted, packages signed by this publisher will pass trust verification during import. When revoked, the certificate is marked as untrusted.

Parameters

Request Body (application/json)

FieldTypeRequiredDefaultDescription
certPathstringRequired—Path to the .cert.json file on the server filesystem
revokebooleanNofalseSet to true to revoke trust instead of granting it

Response

Returns the standard envelope with data containing the trust operation result:

Example Response (Trust)

{
  "success": true,
  "data": {
    "publisher": "John Doe",
    "organization": "Acme Labs",
    "keyId": "abc123def456",
    "action": "trusted"
  }
}

Example Response (Revoke)

{
  "success": true,
  "data": {
    "publisher": "John Doe",
    "organization": "Acme Labs",
    "keyId": "abc123def456",
    "action": "revoked"
  }
}

Examples

cURL

# Trust a publisher certificate
curl -X POST http://localhost:5555/api/publishers/trust \
  -H "Content-Type: application/json" \
  -d '{"certPath": "C:\\Keys\\JohnDoe.cert.json"}'

# Revoke a publisher certificate
curl -X POST http://localhost:5555/api/publishers/trust \
  -H "Content-Type: application/json" \
  -d '{"certPath": "C:\\Keys\\JohnDoe.cert.json", "revoke": true}'

PowerShell

# Trust a publisher certificate
$body = @{ certPath = "C:\Keys\JohnDoe.cert.json" } | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/publishers/trust `
  -Method POST -ContentType "application/json" -Body $body

# Revoke a publisher certificate
$body = @{
    certPath = "C:\Keys\JohnDoe.cert.json"
    revoke   = $true
} | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/publishers/trust `
  -Method POST -ContentType "application/json" -Body $body

Error Conditions

  • certPath not provided → 400
  • Certificate file not found → 400
  • Invalid certificate file format → 400
