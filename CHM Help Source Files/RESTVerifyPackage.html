REST API: POST /api/packages/verify
Library Manager for Venus 6 » REST API Reference » Verify Package

Endpoint

POST /api/packages/verify

Description

Verifies the integrity and signature of a .hxlibpkg or .hxlibarch package without importing it. Equivalent to the CLI command verify-package. Accepts either a file upload (multipart) or a server-side file path.

Parameters

Request Body (multipart/form-data)

You can provide either a file upload or a server-side path, but not both:

FieldTypeRequiredDescription
packagefileNo*The .hxlibpkg or .hxlibarch file to verify
filePathstringNo*Server-side path to the package file

* Either package (file upload) or filePath (server path) must be provided.

Response

Returns the standard envelope with data containing an array of VerificationResult objects (one per package in the file; archives may contain multiple):

FieldTypeDescription
packagestringPackage filename
signedbooleanWhether the package contains a signature
validbooleanWhether the HMAC-SHA256 signature is valid
code_signedbooleanWhether the package has an Ed25519 code signature
publisher_certobject|nullPublisher certificate details (if code-signed)
oem_verifiedbooleanWhether the publisher matches a known OEM or laboratory automation company name
errorsstring[]Verification errors
warningsstring[]Verification warnings

Example Response

{
  "success": true,
  "data": [
    {
      "package": "MyLibrary.hxlibpkg",
      "signed": true,
      "valid": true,
      "code_signed": true,
      "publisher_cert": {
        "publisher": "John Doe",
        "organization": "Acme Labs",
        "key_id": "abc123def456",
        "fingerprint": "SHA256:abcdef..."
      },
      "oem_verified": false,
      "errors": [],
      "warnings": []
    }
  ]
}

Examples

cURL

# Verify via file upload
curl -X POST http://localhost:5555/api/packages/verify \
  -F "package=@MyLibrary.hxlibpkg"

# Verify via server-side path
curl -X POST http://localhost:5555/api/packages/verify \
  -F "filePath=C:\\Packages\\MyLibrary.hxlibpkg"

PowerShell

# Verify via file upload
$form = @{ package = Get-Item ".\MyLibrary.hxlibpkg" }
Invoke-RestMethod -Uri http://localhost:5555/api/packages/verify `
  -Method POST -Form $form

# Verify via server-side path
$form = @{ filePath = "C:\Packages\MyLibrary.hxlibpkg" }
Invoke-RestMethod -Uri http://localhost:5555/api/packages/verify `
  -Method POST -Form $form

Error Conditions

  • Neither package nor filePath provided → 400
  • Server-side file not found → 400
  • Invalid package format (not a valid ZIP) → 400
