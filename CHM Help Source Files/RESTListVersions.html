REST API: GET /api/libraries/:name/versions
Library Manager for Venus 6 » REST API Reference » List Versions

Endpoint

GET /api/libraries/:name/versions

Description

Returns an array of cached package versions for a specific library. Equivalent to the CLI command list-versions. These cached versions are stored in the package store and can be used for rollback operations.

Parameters

Path Parameters

ParameterTypeRequiredDescription
namestringRequiredLibrary name

Response

Returns the standard envelope with data containing an array of CachedVersion objects:

FieldTypeDescription
filestringCached package filename
versionstringPackage version string
authorstringPackage author
createdstringOriginal creation date
cachedstringISO 8601 timestamp when the package was cached
sizeintegerFile size in bytes
fullPathstringFull filesystem path to the cached package
