REST API: POST /api/packages/create
Library Manager for Venus 6 » REST API Reference » Create Package

Endpoint

POST /api/packages/create

Description

Creates a .hxlibpkg package from a JSON specification file. Equivalent to the CLI command create-package. The spec file and all files it references must exist on the server filesystem. Optionally signs the package with an Ed25519 private key and publisher certificate.

This endpoint is serialized by an internal mutex to prevent concurrent database writes.

Parameters

Request Body (application/json)

FieldTypeRequiredDescription
specPathstringRequiredPath to the JSON spec file on the server filesystem
outputstringRequiredOutput path for the .hxlibpkg file on the server
signKeystringNoPath to Ed25519 private key PEM file for code signing
signCertstringNoPath to publisher .cert.json file for code signing
authorPasswordstringNoOEM author password (required for restricted author names)

Response

Returns the standard envelope with data containing the package creation result:

Example Response

{
  "success": true,
  "data": {
    "outputPath": "C:\\Packages\\MyLibrary.hxlibpkg",
    "libraryName": "MyLibrary",
    "version": "2.1.0",
    "author": "John Doe",
    "libraryFiles": 5,
    "demoFiles": 2,
    "codeSigned": true,
    "publisher": "John Doe",
    "size": 245760
  }
}

Examples

cURL

# Create an unsigned package
curl -X POST http://localhost:5555/api/packages/create \
  -H "Content-Type: application/json" \
  -d '{
    "specPath": "C:\\MyLib\\package-spec.json",
    "output": "C:\\Packages\\MyLibrary.hxlibpkg"
  }'

# Create a signed package
curl -X POST http://localhost:5555/api/packages/create \
  -H "Content-Type: application/json" \
  -d '{
    "specPath": "C:\\MyLib\\package-spec.json",
    "output": "C:\\Packages\\MyLibrary.hxlibpkg",
    "signKey": "C:\\Keys\\private.pem",
    "signCert": "C:\\Keys\\publisher.cert.json"
  }'

PowerShell

# Create an unsigned package
$body = @{
    specPath = "C:\MyLib\package-spec.json"
    output   = "C:\Packages\MyLibrary.hxlibpkg"
} | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/packages/create `
  -Method POST -ContentType "application/json" -Body $body

# Create a signed package
$body = @{
    specPath = "C:\MyLib\package-spec.json"
    output   = "C:\Packages\MyLibrary.hxlibpkg"
    signKey  = "C:\Keys\private.pem"
    signCert = "C:\Keys\publisher.cert.json"
} | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/packages/create `
  -Method POST -ContentType "application/json" -Body $body

Error Conditions

  • specPath not provided → 400
  • output not provided → 400
  • Spec file does not exist → 400
  • Invalid spec file contents → 400
  • Referenced library files not found → 400
  • Sign key provided without cert (or vice versa) → 400
  • Restricted OEM author without authorPassword → 400
