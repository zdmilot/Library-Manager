REST API: GET /api/system-libraries/verify
Library Manager for Venus 6 » REST API Reference » Verify System Library Hashes

Endpoint

GET /api/system-libraries/verify

Description

Verifies the integrity of Hamilton system libraries by comparing current file hashes against the known-good baseline. Equivalent to the CLI command verify-syslib-hashes. Returns categorized results showing which files are unchanged, tampered, missing, or encountered errors.

Parameters

Query Parameters

ParameterTypeRequiredDescription
hashFilestringNoOverride the path to the baseline hash file. Default: uses the application's built-in baseline.
libDirstringNoOverride the library directory to verify. Default: from DB settings, fallback C:\Program Files (x86)\HAMILTON\Library

Response

Returns the standard envelope with data containing a SyslibVerifyResult object:

FieldTypeDescription
okobject[]Files that match the baseline hash (integrity verified)
tamperedobject[]Files whose hash differs from the baseline (potentially modified)
missingobject[]Files present in the baseline but not found on disk
errorsobject[]Files that could not be read or hashed due to errors

Example Response

{
  "success": true,
  "data": {
    "ok": [
      {
        "file": "HSLExtensions\\HSLExtensions.hsl",
        "expected": "sha256:abc123...",
        "actual": "sha256:abc123..."
      }
    ],
    "tampered": [
      {
        "file": "HSLTipCountingLib\\HSLTipCountingLib.hsl",
        "expected": "sha256:def456...",
        "actual": "sha256:789ghi..."
      }
    ],
    "missing": [
      {
        "file": "HSLOldLib\\HSLOldLib.hsl",
        "expected": "sha256:jkl012..."
      }
    ],
    "errors": []
  }
}

Examples

cURL

# Verify with default settings
curl http://localhost:5555/api/system-libraries/verify

# Verify with custom paths
curl "http://localhost:5555/api/system-libraries/verify?hashFile=C%3A%5CBaseline%5Chashes.json&libDir=D%3A%5CHamilton%5CLibrary"

PowerShell

# Verify with default settings
Invoke-RestMethod http://localhost:5555/api/system-libraries/verify

# Verify with custom paths
$params = @{
    hashFile = "C:\Baseline\hashes.json"
    libDir   = "D:\Hamilton\Library"
}
$query = ($params.GetEnumerator() | ForEach-Object { "$($_.Key)=$([uri]::EscapeDataString($_.Value))" }) -join "&"
Invoke-RestMethod "http://localhost:5555/api/system-libraries/verify?$query"
