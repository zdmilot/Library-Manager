REST API: GET /api/libraries/:nameOrId
Library Manager for Venus 6 » REST API Reference » Get Library

Endpoint

GET /api/libraries/:nameOrId

Description

Returns details for a single installed library, looked up by its name or internal database ID. Returns a 404 error if the library is not found.

Parameters

Path Parameters

ParameterTypeRequiredDescription
nameOrIdstringRequiredLibrary name (e.g. MyLibrary) or internal database _id

Response

Returns the standard envelope with data containing a single Library object. See List Libraries for the full Library schema.

Error Response (404)

{
  "success": false,
  "error": "Library 'UnknownLib' not found"
}

Examples

cURL

# Get by name
curl http://localhost:5555/api/libraries/MyLibrary

# Get by internal ID
curl http://localhost:5555/api/libraries/abc123

PowerShell

# Get by name
Invoke-RestMethod http://localhost:5555/api/libraries/MyLibrary

# Get by internal ID
Invoke-RestMethod http://localhost:5555/api/libraries/abc123
