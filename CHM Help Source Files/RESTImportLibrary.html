REST API: POST /api/libraries/import
Library Manager for Venus 6 » REST API Reference » Import Library

Endpoint

POST /api/libraries/import

Description

Imports a single .hxlibpkg library package via multipart file upload. Equivalent to the CLI command import-lib. The server verifies the package signature, validates the manifest, extracts files to disk, computes integrity hashes, parses public HSL functions, and registers the library in the database.

This endpoint is serialized by an internal mutex to prevent concurrent database writes.

Parameters

Request Body (multipart/form-data)

FieldTypeRequiredDescription
packagefileRequiredThe .hxlibpkg file to import (max 500 MB)
forcestringNo"true" to overwrite an already-installed library and override signature verification failure
noGroupstringNo"true" to skip auto-assigning the library to a group
noCachestringNo"true" to skip caching the package in the package store
authorPasswordstringNoPassword to authorize importing packages with a restricted OEM or laboratory automation company author name
requireTruststringNo"true" to require the package publisher to be trusted

Response

Returns the standard envelope with data containing an ImportResult object:

FieldTypeDescription
libraryNamestringName of the imported library
versionstringLibrary version
authorstringPackage author
filesExtractedintegerNumber of files extracted to disk
libInstallPathstringLibrary install directory
demoInstallPathstringDemo methods install directory
cachedPathstring|nullPackage store cache path (null if caching was skipped)
signatureStatusstring"valid", "failed", or "unsigned"
comDllsstring[]COM DLLs that require manual registration

The response may also include a warnings array with version mismatch notices or signature warnings.

Example Response

{
  "success": true,
  "data": {
    "libraryName": "MyLibrary",
    "version": "2.1.0",
    "author": "John Doe",
    "filesExtracted": 12,
    "libInstallPath": "C:\\Program Files (x86)\\HAMILTON\\Library\\MyLibrary",
    "demoInstallPath": "C:\\Program Files (x86)\\HAMILTON\\Methods\\Library Demo Methods\\MyLibrary",
    "cachedPath": "C:\\ProgramData\\Library Manager\\packages\\MyLibrary\\MyLibrary_2.1.0.hxlibpkg",
    "signatureStatus": "valid",

  },
  "warnings": []
}

Examples

cURL

# Basic import
curl -X POST http://localhost:5555/api/libraries/import \
  -F "package=@MyLibrary.hxlibpkg"

# Force overwrite with no caching
curl -X POST http://localhost:5555/api/libraries/import \
  -F "package=@MyLibrary.hxlibpkg" \
  -F "force=true" \
  -F "noCache=true"

# Import with OEM author password
curl -X POST http://localhost:5555/api/libraries/import \
  -F "package=@OEMLibrary.hxlibpkg" \
  -F "authorPassword=secret123"

PowerShell

# Basic import
$form = @{ package = Get-Item ".\MyLibrary.hxlibpkg" }
Invoke-RestMethod -Uri http://localhost:5555/api/libraries/import `
  -Method POST -Form $form

# Force overwrite
$form = @{
    package = Get-Item ".\MyLibrary.hxlibpkg"
    force   = "true"
}
Invoke-RestMethod -Uri http://localhost:5555/api/libraries/import `
  -Method POST -Form $form

Error Conditions

  • No package file uploaded → 400
  • Invalid package (no manifest.json) → 400
  • Library already installed without force=true → 400
  • Signature verification failed without force=true → 400
  • Restricted OEM author without authorPassword → 400
  • Untrusted publisher with requireTrust=true → 400
