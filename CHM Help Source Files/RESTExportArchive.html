REST API: POST /api/libraries/export-archive
Library Manager for Venus 6 » REST API Reference » Export Archive

Endpoint

POST /api/libraries/export-archive

Description

Exports one or more installed libraries as a downloadable binary .hxlibarch archive file. Equivalent to the CLI command export-archive. You can export all non-system libraries, or specify specific libraries by name or ID. The response is streamed as application/octet-stream.

Parameters

Request Body (application/json)

FieldTypeRequiredDescription
allbooleanNoExport all non-system libraries. Overrides names and ids.
namesstring[]NoArray of library names to export
idsstring[]NoArray of library database IDs to export

At least one of all, names, or ids must be provided.

Response

On success, the response is a binary .hxlibarch file with the following headers:

HeaderValue
Content-Typeapplication/octet-stream
Content-Dispositionattachment; filename="libraries.hxlibarch"

On error, returns a JSON error response.

Error Response

{
  "success": false,
  "error": "No libraries specified for export"
}

Examples

cURL

# Export all libraries
curl -X POST http://localhost:5555/api/libraries/export-archive \
  -H "Content-Type: application/json" \
  -d '{"all": true}' \
  --output all_libraries.hxlibarch

# Export specific libraries by name
curl -X POST http://localhost:5555/api/libraries/export-archive \
  -H "Content-Type: application/json" \
  -d '{"names": ["LibraryA", "LibraryB"]}' \
  --output selected_libraries.hxlibarch

# Export by IDs
curl -X POST http://localhost:5555/api/libraries/export-archive \
  -H "Content-Type: application/json" \
  -d '{"ids": ["abc123", "def456"]}' \
  --output selected_libraries.hxlibarch

PowerShell

# Export all libraries
$body = @{ all = $true } | ConvertTo-Json
Invoke-WebRequest -Uri http://localhost:5555/api/libraries/export-archive `
  -Method POST -ContentType "application/json" -Body $body `
  -OutFile ".\all_libraries.hxlibarch"

# Export specific libraries by name
$body = @{ names = @("LibraryA", "LibraryB") } | ConvertTo-Json
Invoke-WebRequest -Uri http://localhost:5555/api/libraries/export-archive `
  -Method POST -ContentType "application/json" -Body $body `
  -OutFile ".\selected_libraries.hxlibarch"

Error Conditions

  • No libraries specified (all, names, and ids all absent) → 400
  • Named or ID-referenced library not found → 400
  • Archive creation failure → 400
