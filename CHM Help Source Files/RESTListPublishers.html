REST API: GET /api/publishers
Library Manager for Venus 6 » REST API Reference » List Publishers

Endpoint

GET /api/publishers

Description

Returns an array of registered publisher signing certificates. Each publisher entry includes its name and an array of associated certificates with key identifiers and fingerprints.

Parameters

This endpoint takes no parameters.

Response

Returns the standard envelope with data containing an array of Publisher objects:

FieldTypeDescription
namestringPublisher name
certificatesobject[]Array of certificate objects

Certificate Object

FieldTypeDescription
publisherstringPublisher name
organizationstringOrganization name
key_idstringEd25519 key identifier
fingerprintstringCertificate fingerprint
created_datestringISO 8601 certificate creation date

Example Response

{
  "success": true,
  "data": [
    {
      "name": "John Doe",
      "certificates": [
        {
          "publisher": "John Doe",
          "organization": "Acme Labs",
          "key_id": "abc123def456",
          "fingerprint": "SHA256:abcdef1234567890...",
          "created_date": "2026-01-01T00:00:00.000Z"
        }
      ]
    }
  ]
}
