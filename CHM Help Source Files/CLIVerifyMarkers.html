CLI: verify-markers
Library Manager » CLI Reference » verify-markers

Description

Verifies the integrity of .libmgr marker files for installed libraries. Each non-system library managed by Library Manager has a .libmgr marker file in its install directory that records file hashes at install time. This command checks that each marker exists and that all recorded file hashes still match the files on disk, detecting any post-installation tampering or accidental modification.

Usage

node cli.js verify-markers [--name <name>] [--json]

Options

FlagRequiredDescription
--name <name>NoVerify a single library by name. If omitted, all non-system installed libraries are verified.
--jsonNoPrint machine-readable JSON output instead of human-readable text

Verification Process

  1. All installed libraries are loaded from the database (soft-deleted and system libraries are excluded)
  2. For each library, the tool checks for a .libmgr marker file in the library’s install directory
  3. If the marker exists, recorded file hashes are compared against the current files on disk
  4. Results are categorized as: ok (all hashes match), no_marker (marker file missing), invalid (hash mismatch or missing files), or no_path (no install path recorded)

Possible Outcomes

StatusMeaning
okMarker exists and all file hashes match — library is intact
no_markerNo .libmgr marker file found in the install directory (library may have been installed before markers were introduced, or the marker was deleted)
invalidMarker exists but one or more files are missing or have been modified since installation
no_pathThe library’s database record has no install path (unusual — may indicate a corrupt database entry)

Examples

:: Verify all installed libraries
node cli.js verify-markers

:: Verify a single library
node cli.js verify-markers --name "MyLibrary"

:: Machine-readable output for scripting
node cli.js verify-markers --json

JSON Output Format

When --json is specified, the output is a JSON array of result objects:

[
  {
    "library": "MyLibrary",
    "status": "ok",
    "path": "C:\\Program Files (x86)\\HAMILTON\\Library\\MyLibrary",
    "version": "1.0.0"
  },
  {
    "library": "OtherLib",
    "status": "no_marker",
    "message": "No .libmgr marker found",
    "path": "C:\\Program Files (x86)\\HAMILTON\\Library\\OtherLib"
  },
  {
    "library": "ModifiedLib",
    "status": "invalid",
    "path": "C:\\Program Files (x86)\\HAMILTON\\Library\\ModifiedLib",
    "missing": ["helper.hs_"],
    "tampered": ["main.hsl"],
    "errors": []
  }
]
