REST API: GET /api/libraries/:nameOrId/export
Library Manager for Venus 6 » REST API Reference » Export Library

Endpoint

GET /api/libraries/:nameOrId/export

Description

Exports an installed library as a downloadable binary .hxlibpkg package file. Equivalent to the CLI command export-lib. The response is streamed directly as application/octet-stream with a Content-Disposition header for file download.

Parameters

Path Parameters

ParameterTypeRequiredDescription
nameOrIdstringRequiredLibrary name (e.g. MyLibrary) or internal database _id

Response

On success, the response is a binary .hxlibpkg file with the following headers:

HeaderValue
Content-Typeapplication/octet-stream
Content-Dispositionattachment; filename="LibraryName.hxlibpkg"

On error, returns a JSON error response.

Error Response (404)

{
  "success": false,
  "error": "Library 'UnknownLib' not found"
}

Examples

cURL

# Export by name and save to file
curl http://localhost:5555/api/libraries/MyLibrary/export \
  --output MyLibrary.hxlibpkg

# Export by ID
curl http://localhost:5555/api/libraries/abc123/export \
  --output MyLibrary.hxlibpkg

PowerShell

# Export by name
Invoke-WebRequest -Uri http://localhost:5555/api/libraries/MyLibrary/export `
  -OutFile ".\MyLibrary.hxlibpkg"

# Export by ID
Invoke-WebRequest -Uri http://localhost:5555/api/libraries/abc123/export `
  -OutFile ".\MyLibrary.hxlibpkg"

Error Conditions

  • Library not found → 404
  • Export packaging fails → 400
