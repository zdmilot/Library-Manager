REST API: GET /api/audit
Library Manager for Venus 6 » REST API Reference » Get Audit Trail

Endpoint

GET /api/audit

Description

Retrieves the library management audit trail log. All import, export, delete, rollback, and other management operations are recorded with timestamps, usernames, and system information. The audit trail is shared across CLI, REST API, and COM interfaces.

Parameters

Query Parameters

ParameterTypeRequiredDefaultDescription
limitintegerNoAll entriesReturn only the last N entries

Response

Returns the standard envelope with data containing an array of AuditEntry objects:

FieldTypeDescription
eventstringEvent type (e.g. "import", "delete", "rollback", "export")
timestampstringISO 8601 event timestamp
usernamestringWindows username that performed the operation
windows_versionstringWindows version at time of operation
venus_versionstringVENUS version (if detected)
hostnamestringMachine hostname
detailsobjectEvent-specific details (library name, version, paths, etc.)

Examples

cURL

# Get full audit trail
curl http://localhost:5555/api/audit

# Get last 10 entries
curl "http://localhost:5555/api/audit?limit=10"

PowerShell

# Get full audit trail
Invoke-RestMethod http://localhost:5555/api/audit

# Get last 10 entries
Invoke-RestMethod "http://localhost:5555/api/audit?limit=10"
