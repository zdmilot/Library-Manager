REST API: GET /api/health
Library Manager for Venus 6 » REST API Reference » Health Check

Endpoint

GET /api/health

Description

Returns the current server status, application version, uptime in seconds, hostname, operating system platform, and Node.js version. Use this endpoint to confirm the REST API server is running and reachable.

Parameters

This endpoint takes no parameters.

Response

Returns a JSON object (not wrapped in the standard success/data envelope):

FieldTypeDescription
statusstringAlways "ok" when the server is running
versionstringApplication version (e.g. "1.6.5")
uptimenumberServer uptime in seconds
hostnamestringMachine hostname
platformstringOperating system platform (e.g. "win32")
nodeVersionstringNode.js runtime version (e.g. "v18.17.0")

Example Response

{
  "status": "ok",
  "version": "1.6.5",
  "uptime": 3421.87,
  "hostname": "LAB-PC-01",
  "platform": "win32",
  "nodeVersion": "v18.17.0"
}

Examples

cURL

curl http://localhost:5555/api/health

PowerShell

Invoke-RestMethod http://localhost:5555/api/health
