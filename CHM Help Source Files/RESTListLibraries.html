REST API: GET /api/libraries
Library Manager for Venus 6 » REST API Reference » List Libraries

Endpoint

GET /api/libraries

Description

Returns an array of all installed libraries. Equivalent to the CLI command list-libs. By default, soft-deleted libraries are excluded from results.

Parameters

Query Parameters

ParameterTypeRequiredDefaultDescription
includeDeletedbooleanNofalseWhen true, include soft-deleted libraries in the results

Response

Returns the standard envelope with data containing an array of Library objects:

FieldTypeDescription
_idstringUnique library ID
library_namestringLibrary name
authorstringAuthor name
organizationstringOrganization name
versionstringVersion string
venus_compatibilitystringVENUS version compatibility
descriptionstringLibrary description
github_urlstringGitHub repository URL
tagsstring[]Library tags
created_datestringISO 8601 creation date
library_filesstring[]Installed library file paths
demo_method_filesstring[]Installed demo method file paths
help_filesstring[]Associated help file paths
com_register_dllsstring[]COM DLLs requiring registration
lib_install_pathstringLibrary install directory
demo_install_pathstringDemo methods install directory
installed_datestringISO 8601 install timestamp
installed_bystringWindows username that performed the install
public_functionsobject[]Parsed public HSL functions
deletedbooleanSoft-delete flag
deleted_datestringISO 8601 deletion timestamp (if deleted)

Examples

cURL

# List all active libraries
curl http://localhost:5555/api/libraries

# Include soft-deleted libraries
curl "http://localhost:5555/api/libraries?includeDeleted=true"

PowerShell

# List all active libraries
Invoke-RestMethod http://localhost:5555/api/libraries

# Include soft-deleted libraries
Invoke-RestMethod "http://localhost:5555/api/libraries?includeDeleted=true"
