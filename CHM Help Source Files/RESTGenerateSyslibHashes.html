REST API: POST /api/system-libraries/generate-hashes
Library Manager for Venus 6 » REST API Reference » Generate System Library Hashes

Endpoint

POST /api/system-libraries/generate-hashes

Description

Generates a baseline integrity hash file for Hamilton system libraries from a known-good library directory. Equivalent to the CLI command generate-syslib-hashes. The generated baseline is used by the verify endpoint to detect tampered or modified files.

Parameters

Request Body (application/json)

FieldTypeRequiredDescription
sourceDirstringRequiredPath to the known-good Library folder on the server
outputstringNoOutput path for the baseline hash file. Default: stored in the application data directory.

Response

Returns the standard envelope with data containing the generation result:

Example Response

{
  "success": true,
  "data": {
    "sourceDir": "C:\\Program Files (x86)\\HAMILTON\\Library",
    "outputPath": "C:\\ProgramData\\Library Manager\\system_library_hashes.json",
    "filesHashed": 142,
    "librariesProcessed": 38
  }
}

Examples

cURL

# Generate with default output location
curl -X POST http://localhost:5555/api/system-libraries/generate-hashes \
  -H "Content-Type: application/json" \
  -d '{"sourceDir": "C:\\Program Files (x86)\\HAMILTON\\Library"}'

# Generate with custom output path
curl -X POST http://localhost:5555/api/system-libraries/generate-hashes \
  -H "Content-Type: application/json" \
  -d '{
    "sourceDir": "C:\\Program Files (x86)\\HAMILTON\\Library",
    "output": "C:\\Baseline\\syslib_hashes.json"
  }'

PowerShell

# Generate with default output location
$body = @{
    sourceDir = "C:\Program Files (x86)\HAMILTON\Library"
} | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/system-libraries/generate-hashes `
  -Method POST -ContentType "application/json" -Body $body

# Generate with custom output path
$body = @{
    sourceDir = "C:\Program Files (x86)\HAMILTON\Library"
    output    = "C:\Baseline\syslib_hashes.json"
} | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:5555/api/system-libraries/generate-hashes `
  -Method POST -ContentType "application/json" -Body $body

Error Conditions

  • sourceDir not provided → 400
  • Source directory does not exist → 400
  • No system library files found in the source directory → 400
  • Output path not writable → 400
