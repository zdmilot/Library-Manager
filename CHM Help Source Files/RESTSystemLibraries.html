REST API: GET /api/system-libraries
Library Manager for Venus 6 » REST API Reference » System Libraries

Endpoint

GET /api/system-libraries

Description

Returns an array of Hamilton built-in system library definitions. System libraries are pre-installed with VENUS 6 and are tracked separately from user-installed libraries. This data is used for integrity verification and reference.

Parameters

This endpoint takes no parameters.

Response

Returns the standard envelope with data containing an array of system library objects:

Example Response

{
  "success": true,
  "data": [
    {
      "library_name": "HSLExtensions",
      "description": "Hamilton Standard Language Extensions",
      "files": [
        "HSLExtensions.hsl",
        "HSLExtensions.hs_"
      ],
      "category": "core"
    },
    {
      "library_name": "HSLTipCountingLib",
      "description": "Tip counting and tracking library",
      "files": [
        "HSLTipCountingLib.hsl",
        "HSLTipCountingLib.hs_"
      ],
      "category": "liquid-handling"
    }
  ]
}

Examples

cURL

curl http://localhost:5555/api/system-libraries

PowerShell

Invoke-RestMethod http://localhost:5555/api/system-libraries
