REST API: GET /api/settings
Library Manager for Venus 6 » REST API Reference » Get Settings

Endpoint

GET /api/settings

Description

Retrieves the current application settings. Returns configuration values such as file paths, directory overrides, and application preferences.

Parameters

This endpoint takes no parameters.

Response

Returns the standard envelope with data containing the application settings object. The exact shape depends on the current configuration:

Example Response

{
  "success": true,
  "data": {
    "lib_dir": "C:\\Program Files (x86)\\HAMILTON\\Library",
    "met_dir": "C:\\Program Files (x86)\\HAMILTON\\Methods",
    "store_dir": "C:\\ProgramData\\Library Manager\\packages",
    "db_path": "C:\\ProgramData\\Library Manager",
    "venus_version": "6.3",
    "theme": "light",
    "show_deleted": false,
    "auto_group": true
  }
}

Examples

cURL

curl http://localhost:5555/api/settings

PowerShell

Invoke-RestMethod http://localhost:5555/api/settings
